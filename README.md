## 📝 | Tutorial

### 💪🏻 Instruct
> Tệp `config.js` Phải Được Cấu Hình Trước
> File `.env` Thay Đổi `BOT_TOKEN=<TOKEN CỦA BẠN>` Và `BOT_ID=<CLIENT ID BOT CỦA BẠN>`

Các Lệnh Setup Và Khởi Động Bot
```sh
npm install
npm run deploy
```
Start the bot
```sh
node index.js
```
